式と文
～ 非公式 日本語リファレンス ～


式

プログラミングにおける 式(= expression) とは、"なんらかの[評価した値]を返すもの" のことを指します。
また、"評価" とは、関数などの式や不等式・if式などの各種計算式を "実際に計算すること" を指します。

式は評価をすることで値になるわけなので、当然、値を書き込めるところには式も書き込むことができます。
例えばF#では if は式であるため、次のように記述することが可能です。
let condition = true
let value = if condition then 100 else 500
また、不等式は評価されるとbool型の値となることはよく知られています。
let result = 100 < 200   // true

このように100や"Sample"などの値を書き込めるところであればどこにでも式を書き込むことができます。
これが次節の文と大きく異なる点ですので、違いをしっかりと把握しておきましょう。


文

式とは異なり、評価されても値を返さないものを 文(= statement) と言います。
文は通常、1つ以上の式で構成されます。

F#では、トップレベルでの宣言のみが文で、他はすべて式となっています。
C#やVB.NETでは、ifやfor/foreachなどの各種制御文がありますが、それらは文字どおり文です。
F#にも同等の機能が存在していますが、こちらはすべて式となっています。

F#で文となるものとしてはopen宣言やnamespace宣言、module宣言、type宣言などがあります。
