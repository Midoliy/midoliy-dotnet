値の束縛

束縛 または バインド とは、値 と 名称 を関連付けることを指します。また、この関連付けのことを バインディング と呼ぶこともあります。

F# では 値の束縛 をする際には let というキーワードを使用します。この 束縛 を特に let束縛 と呼びます。
F# の let束縛 は以下のようにすることで、値 と 名称 を関連づけることが可能です。
// int型の 値 を age と言う名称で束縛
let age = 30

// string型の 値 を name と言う名称で束縛
let name = "midoliy"
